## Constraints

- Preserve the existing Astro static-site structure.
- Keep new content metadata optional unless a deliberate migration is chosen.
- Do not break existing records, series routes, tags, search, RSS, SEO, dark mode, or mobile navigation.
- Keep 만냥구름 as warmth and observation, not a site-wide gimmick.
- Keep all claims bounded to the evidence surface that proves them.

- `https://ji-yong.com` was smoke-checked on 2026-07-04 and failed required persona-text checks, so current live-domain evidence does not prove this branch is deployed.
- Deploy trigger is external to this repo's CI workflow: current evidence shows Vercel creates Production deployments from the default branch `master`. Merging or pushing this persona branch to that default branch is expected to trigger Vercel, but the resulting deployment still needs production-smoke verification.

- Local checks prove local behavior only; production readiness still needs deploy-specific verification.
- JiYong's human approval is recorded for the current PR scope on 2026-07-04; any future broad content migration or materially different design direction would need a new review.
